## 模块名称
riscv_alu (RISC-V Arithmetic Logic Unit)

## 逻辑类型
组合逻辑（子模块）

注意：作为子模块，其 process() 调用由顶层模块（riscv_exec）触发，在单个周期内完成计算。
不需要 clk 输入端口和时钟边沿检测逻辑。

## 功能概述
RISC-V ALU 模块执行算术和逻辑运算，包括加法、减法、按位与/或/异或、左移/右移（逻辑/算术）、无符号/有符号比较。该模块是纯组合逻辑，输出在输入稳定后立即有效。被 riscv_exec 主执行单元调用。

## 端口定义

### 输入信号
| 端口名 | 位宽 | 方向 | 描述 |
|--------|------|------|------|
| alu_a_i | 31:0 | input | ALU 操作数 A |
| alu_b_i | 31:0 | input | ALU 操作数 B |
| alu_op_i | 3:0 | input | ALU 操作码 |

### 输出信号
| 端口名 | 位宽 | 方向 | 描述 |
|--------|------|------|------|
| alu_p_o | 31:0 | output | ALU 运算结果 |

## 功能说明

### 1. 操作码解码
ALU 根据 alu_op_i 输入选择运算类型：
- ALU_NONE (4'b0000): 直通，输出 alu_a_i
- ALU_SHIFTL (4'b0001): 逻辑左移，根据 alu_b_i 的低位进行桶形移位
- ALU_SHIFTR (4'b0010): 逻辑右移
- ALU_SHIFTR_ARITH (4'b0011): 算术右移（符号扩展）
- ALU_ADD (4'b0100): 加法
- ALU_SUB (4'b0110): 减法
- ALU_AND (4'b0111): 按位与
- ALU_OR (4'b1000): 按位或
- ALU_XOR (4'b1001): 按位异或
- ALU_LESS_THAN (4'b1010): 无符号小于比较
- ALU_LESS_THAN_SIGNED (4'b1011): 有符号小于比较

### 2. 桶形移位器实现
左移和右移使用五级桶形移位器（1/2/4/8/16 位级联）实现：
- 移位量由 alu_b_i 的低 5 位决定
- 每级移位器根据对应位是否置位决定是否移位
- 算术右移时，高位填充符号位（alu_a_i 的 MSB）
- 逻辑右移时，高位填充 0

### 3. 比较操作
- ALU_LESS_THAN: 输出 1 当 (alu_a_i < alu_b_i) 无符号比较
- ALU_LESS_THAN_SIGNED: 输出 1 当有符号比较结果为真

### 4. 输出
- 结果通过 alu_p_o 输出
- 始终有一个有效结果（无需 valid 信号）
